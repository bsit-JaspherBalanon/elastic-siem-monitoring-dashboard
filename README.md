# SIEM-Based Security Event Monitoring Dashboard

## Overview

This project demonstrates the implementation of a Security Information and Event Management (SIEM) solution using the Elastic Stack. The system collects, processes, stores, and visualizes log data from monitored systems to support security monitoring and incident detection.

## Features

* Real-time log collection using Filebeat
* Centralized log storage in Elasticsearch
* Interactive dashboards in Kibana
* Security event monitoring and visualization
* Event trend analysis
* Host activity monitoring
* Log categorization and filtering

## Technologies Used

* Elasticsearch 8.19
* Kibana 8.19
* Filebeat 8.19
* Ubuntu Server
* Linux System Logs

## Dashboard Components

### Total Security Events

Displays the total number of collected log events.

### Event Trends

Visualizes log activity over time.

### Event Categories

Shows the distribution of collected log types.

### Host Activity

Displays log events generated by monitored hosts.

## Architecture

System Logs → Filebeat → Elasticsearch → Kibana Dashboard
