# Deployment Notes

## Environment

- Ubuntu Server 24.04
- Elasticsearch 8.19
- Kibana 8.19
- Filebeat 8.19

## Components

### Elasticsearch
Stores and indexes collected log data.

### Filebeat
Collects Linux system logs and forwards them to Elasticsearch.

### Kibana
Provides dashboards, visualizations, and log analysis capabilities.

## Security Monitoring Features

- Log Collection
- Authentication Event Monitoring
- Event Visualization
- Host Activity Monitoring
- Security Event Analysis
